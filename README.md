# Stroke-Risk-Prediction
This project uses patterns from a real-world stroke prediction dataset to explore how demographic, lifestyle, and clinical factors relate to stroke outcomes. Built for learning and demonstration — not a substitute for medical advice.<br>
What the model considers<br>
-> Demographics: gender, age, marriage status<br>
-> Clinical: hypertension, heart disease, glucose, BMI<br>
-> Lifestyle: work type, residence, smoking status<br>
